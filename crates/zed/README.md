# css-modules-kit-zed

The Zed extension for CSS Modules Kit

## Installation

1. Install "CSS Modules Kit" extension on Zed.
2. Add the following to your `~./config/zed/settings.json` file:
   ```json
   {
     "languages": {
       "CSS": {
         "language_servers": ["vtsls", "..."]
       }
     }
   }
   ```
3. Restart Zed.
